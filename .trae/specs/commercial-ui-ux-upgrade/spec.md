# 商业级棋牌视听与交互升级 Spec (对标《欢乐妞妞》)

## Why
通过分析成熟商业棋牌视频（如《欢乐妞妞》等平台级游戏），我们发现当前项目虽然底层逻辑跑通并完成了初步的“发财红”视觉改造，但在**视听沉浸感、赌局氛围营造、以及玩家微交互节奏**上仍有显著差距。商业棋牌的核心在于利用强烈的感官刺激（真人语音、筹码轨迹、心跳倒数、全屏爆分）来延长玩家留存、激发荷尔蒙并促进充值消费。

### 核心区别对比
1. **视听反馈 (Audio & Visual)**
   - **商业级**：全套真人语音报牌（“牛七！”、“牛牛通杀！”），筹码从输家飞向赢家的满屏轨迹，配合局势紧张度切换的 BGM（如下注倒计时的滴答声、开底牌的心跳声）。
   - **当前项目**：仅有单调的“Click”音效和最后结算的红包雨，缺乏过程中的听觉和视觉反馈，显得“干瘪”。
2. **搓牌与悬念 (Suspense & Squeezing)**
   - **商业级**：第五张底牌的揭晓极其隆重，会有 3D 翻转、局部遮罩缓慢滑开的“搓牌”动作，伴随心跳声逐渐加速。
   - **当前项目**：第五张牌仅有一个简单的倾斜 CSS，点击直接翻开，缺乏“执掌全场”的悬念感。
3. **高潮爆发表现 (Mega Win)**
   - **商业级**：当玩家拿到“牛牛”及以上的大牌（3倍/5倍/8倍）时，会有震屏的 `Big Win` / `Mega Win` 弹窗及聚光灯动效。
   - **当前项目**：只是数字跳动，大牌和小牌的结算视觉差异不够强烈。
4. **局内社交互动 (Social Engagement)**
   - **商业级**：玩家之间可以扔番茄、送啤酒、发快捷嘲讽语音，活跃牌桌气氛，这也是极佳的收费点。
   - **当前项目**：玩家之间只能干瞪眼，像在玩单机。

## 完善可落地的参考优化方案 (What Changes)
- **增加筹码飞行动效 (Chip Animations)**: 在结算阶段，引入 `framer-motion` 制作从输家头像飞向庄家，再从庄家飞向赢家的实体筹码轨迹。
- **引入场景音效管理器 (Audio Manager)**: 预埋语音播报接口（发牌声、心跳声、牛牛通杀声），配合前端状态机自动触发。
- **重制搓牌交互 (Card Reveal Polish)**: 第五张牌增加“模糊到清晰”或“缓慢滑动”的 CSS 遮罩过渡，并在看牌时播放心跳音效。
- **高倍数爆分特效 (Big Win)**: 当玩家获得 `bull >= 10` 时，在原有的金币雨基础上，增加屏幕震动（`animate-shake`）和聚光灯爆炸效果。
- **新增快捷互动面板 (Emotes)**: 在头像点击时弹出环形菜单，允许发送基础 Emoji（🍺, 🍅, 💣, 🌹）并附带音效。

## Impact
- Affected specs: 游戏客户端表现层（视听反馈、结算动画、玩家互动）。
- Affected code:
  - `src/App.tsx` (主干流程与动画引擎)
  - `src/components/Card.tsx` (搓牌重制)
  - `src/index.css` (特效动画类)

## ADDED Requirements
### Requirement: 筹码轨迹结算
系统 SHALL 在每局结算（`finished` 状态）时，根据分数正负，在玩家与庄家之间生成带延迟的筹码飞行动画。

#### Scenario: 赢家收钱
- **WHEN** 玩家 A 赢了庄家 1000 分
- **THEN** 结算时，屏幕首先展示庄家飞出筹码到玩家 A 的头像，伴随硬币碰撞的清脆音效。

### Requirement: 大牌震屏特效
系统 SHALL 在玩家获得“牛牛”或“五小牛”等高倍数牌型时，触发高潮视觉反馈。

#### Scenario: 获得牛牛
- **WHEN** 玩家底牌翻开为“牛牛”
- **THEN** 触发 1 秒钟的 `animate-shake` 屏幕震动，播放“牛牛”音效，并在该玩家卡片上叠加闪耀的聚光灯 CSS 滤镜。
