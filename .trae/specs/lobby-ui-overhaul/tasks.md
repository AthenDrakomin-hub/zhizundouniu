# Tasks
- [ ] Task 1: **基础大厅状态结构 (`src/App.tsx`)** - 将 `!isJoined` 状态下的登录表单替换为全屏 `Lobby` 组件。引入 `activeTab` 状态 ('hall', 'room', 'mine') 控制底部三个导航按钮的切换。
- [ ] Task 2: **大厅 Tab 开发 (`src/App.tsx`)** - 
  - 顶部栏：实现带有半透明黑底的头部，左侧显示默认玩家大名（如 BB）和房卡（0张），右侧为商城和 Logo。
  - 中心网格：实现 6 个卡片的 Grid 布局（如 斗牛集合、三公集合、炸金花集合、牌九合集、欢乐斗地主等）。利用 CSS 渐变、暗金色描边和发光效果模拟高档 3D 卡片。
- [ ] Task 3: **建房与我的 Tab 开发 (`src/App.tsx`)** - 
  - 开房 Tab：实现带“加入公会后，等待会长开房”居中提示的空白背景，顶部有“切换”、“刷新”按钮。
  - 我的 Tab：实现带有大头像卡片和下部设置列表（如 聊天室、管理功能、修改手机、房卡包等）的个人中心。
- [ ] Task 4: **游戏弹窗与衔接 (`src/App.tsx`)** - 在大厅点击“斗牛集合”或“开房”时，弹出一个带有毛玻璃质感的对话框（Modal），里面包含“大名”和“房卡/房号”的输入框以及“加入/创建”按钮，连接到 `handleJoin` 进入真实的对局逻辑。
- [ ] Task 5: **管理后台入口绑定 (`src/App.tsx`)** - 在“我的 Tab”的“管理功能”列表中增加一个 Toggle Switch 开关。点击后通过 `window.open` 或路由重定向至 `:3000` (admin-client) 的控制台链接，作为隐蔽的管理入口。

# Task Dependencies
- [Task 1] 为整体大厅导航骨架，必须首先完成。
- [Task 2] 和 [Task 3] 可并行，是核心页面 UI。
- [Task 4] 是连接大厅与当前已有牌桌逻辑的桥梁，在 UI 完成后集成。
- [Task 5] 为锦上添花的功能，作为最后一步完善。
